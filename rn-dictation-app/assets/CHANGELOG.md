# 更新日志 (Changelog)

## [1.0.0] - 2026-04-18
### 新特性 (Feat)
- 🚀 **技术栈重构**：将系统从 Python(NiceGUI) 完全迁移至 React Native (Expo)，提供更原生的移动端体验。
- ✨ **毛玻璃美学**：采用 NativeWind (TailwindCSS) + expo-blur 完美复刻原有的暗黑毛玻璃风格界面。
- 🗣️ **原生发音**：集成 `expo-speech`，实现本地化、无延迟的纯正英语 TTS 朗读。
- 🔐 **安全加固**：使用 `expo-secure-store` 实现了管理员/访客角色权限控制，废弃了原有的硬编码明文密码。
- 💾 **本地化存储**：采用 `AsyncStorage` + Zustand 实现了“应用配置/全局数据”的安全分离存储，不再依赖修改脚本源码。
- 📖 **错题本与流水**：完善了基于时间轴的测验流水记录以及基于错误频次动态衰减的“智能错题本”。
